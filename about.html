What is SpaceGo?

Go (or Baduk or Wei Qi) is an ancient Chinese board game. It's like 4000 years old. The rules are very simple, but the gameplay is very complicated.

Learn to play Go here

SpaceGo specifically is "Go on a Torus". A torus is just a shape that wraps around in all directions - think the old game Asteroids. In practice, it's just Go without edges.

Why it exists

Toroidal Go is already played both in person and online, and even has tournaments. However, I think the UI in this version adds a cool element and makes it easier to wrap your brain around what's really going on across the board.

Besides the potential addition to the Go community, this is a portfolio project.

Tech Used

This is a PlainJS web application hosted on a Google Cloud VM

The back-end is a NodeJS server using Express with a Nginx proxy and a Redis database
